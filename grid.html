Grid System di Bootstrap

Bootstrap mette a disposizione una griglia per l'impaginazione di contenuti.

Questa griglia è composta da 12 colonne; ogni colonna, volendo, può essere suddivisa in altri 12 slot e via così.

Le col stanno SEMPRE dentro le row. Le row possono stare dentro i container, oppure dentro una col esistente.
